# 使用如下： #

    <Calendar
             currentMonthData={this.state.data}
             onScroll={this._onScroll}
             onPick={this._selectDate}
         />

## 参数说明： ##
1. currentMonthData:初始化请求数据
2. onScroll:滚动时回调事件
3. onPick：获取选中日期
